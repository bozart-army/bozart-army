# Bozart Army

Bozart Army is a fictional science-fiction universe set in the year 4780.

This repository serves as a public project space for worldbuilding,
narrative texts, in-world documents, concepts and supporting assets
for a speculative future franchise.

All content is part of an invented, far-future reality or alternative timeline.
There are no references to real-world events or institutions.

## Contents

- Worldbuilding documentation
- Narrative texts and in-world artifacts
- Concepts for factions, systems and technologies
- Supporting material for book, film and series formats

## Fictional Context

All depicted organizations, systems, technologies and conflicts
are entirely fictional and used to explore themes such as power,
control, freedom and humanity within a dystopian future setting.

## Notice

This project is a work of speculative fiction and exists solely
for narrative and artistic purposes.
